# Property Images

This directory contains images for properties, crowdfunding campaigns, and investment opportunities.

## Required Images

Place the following images in this directory:

### Properties
- `property1.jpg` - Property Share (Dolmen REIT)
- `property2.jpg` - Serene Heights Hotel & Resort
- `property3.jpg` - Blue World City
- `property4.jpg` - Capital Smart City
- `property5.jpg` - DHA Phase 8
- `property6.jpg` - Bahria Town Karachi

### Logos
- `logo1.png` - Dolmen REIT Logo
- `logo2.png` - Serene Heights Logo

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1200x800px (3:2 aspect ratio)
- **Max File Size**: 500KB per image
- **Quality**: High quality, professional real estate photography

## Fallback

If images are not available, the PropertyCard component will display a placeholder background color.
